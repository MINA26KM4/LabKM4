== Wymagania

* Java 17+
* Maven
* Docker

== Instalacja

. Sklonuj repozytorium:
+
[source,bash]
----
git clone https://github.com/example/projekt.git
----

. Uruchom aplikację:
+
[source,bash]
----
mvn spring-boot:run
----
